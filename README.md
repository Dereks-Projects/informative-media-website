## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) (v18 or higher)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens a local dev server at `http://localhost:4321`. Auto-refreshes on save.

### Build

```bash
npm run build
```

Generates a static `dist/` folder of production-ready HTML, CSS, and JS. The sitemap is auto-generated at `dist/sitemap-index.xml` during build.

### Preview

```bash
npm run preview
```

Serves the built `dist/` folder locally for final review before deployment.

## CSS Architecture

- **`global.css`** â€” Reset, variables, header, footer, buttons, homepage sections, responsive breakpoints
- **`pages.css`** â€” Reusable inner page styles (detail cards, page headers, content blocks, legal, about, 404)
- **Breakpoints:** 768px (tablet), 1024px (desktop)
- **Approach:** Mobile-first, no inline styles, CSS custom properties for theming

---

## SEO & Analytics

- **Sitemap:** Auto-generated by `@astrojs/sitemap` at build time, submitted to Google Search Console
- **robots.txt:** Allows all crawlers, references sitemap URL
- **Google Analytics 4:** Measurement tag in Layout.astro, runs on all pages
- **Open Graph:** Meta tags for social sharing on every page
